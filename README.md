This project implements a complete Kafka-based system that satisfies all assignment requirements:

- **Kafka Producer & Consumer** - Produces and consumes order messages  
- **Avro Serialization** - All messages use Avro schema with Schema Registry  
- **Real-time Aggregation** - Calculates running average of order prices  
- **Retry Logic** - Handles temporary failures with automatic retry  
- **Dead Letter Queue (DLQ)** - Sends permanently failed messages to DLQ topic  
- **Schema Definition** - Order message schema defined in `order.avsc`
